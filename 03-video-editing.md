# Video Editing Guide (CapCut)

## Why CapCut?

| Criteria | CapCut | Alternatives |
|----------|--------|--------------|
| Cost | Free (Pro: $9/mo) | Varies |
| Learning curve | 1-2 hours | Higher |
| Templates | Excellent | Limited |
| Logo overlay | Yes | Yes |
| Auto-captions | Yes (free) | Often paid |
| Export quality | 4K 60fps | Varies |
| Watermark | None | Some add |

**Bottom line:** Best balance of ease, features, and cost for non-editors.

## Basic Editing Workflow

### Step 1: Import Footage

1. Tap **New Project**
2. Select videos from camera roll
3. Tap **Add** (bottom right)
4. Videos appear on timeline

### Step 2: Trim & Arrange

**To trim a clip:**
1. Tap clip on timeline
2. Drag white handles at ends
3. Remove unwanted parts

**To split a clip:**
1. Position playhead (white line)
2. Tap **Split** button
3. Delete unwanted section

**To reorder:**
- Long press and drag clips

### Step 3: Add Transitions

1. Tap the **white square** between clips
2. Browse transition options
3. Tap to preview, tap again to apply
4. Adjust duration (0.3-0.5s recommended)

**Best for construction:**
- **Basic:** Fade, Dissolve (professional)
- **Reveals:** Wipe, Slide (before/after)
- **Avoid:** Excessive flashy effects (looks unprofessional)

### Step 4: Add Text

1. Tap **Text** at bottom
2. Tap **Add Text**
3. Type your message
4. Style options:
   - Font (use brand font)
   - Color (use brand colors)
   - Size (readable on mobile = LARGE)
   - Position (avoid top/bottom safe zones)

**Text timing:**
- Tap text on timeline
- Drag to position when it appears
- Extend/shorten as needed
- **Minimum 3 seconds** for readability

### Step 5: Add Music

1. Tap **Audio** at bottom
2. Options:
   - **Sounds** - CapCut library (check licensing)
   - **Extract audio** - From other videos
   - **Your music** - Import licensed tracks
3. Tap track, then **Add**
4. Adjust volume: Tap audio clip > Volume slider

**Volume levels:**
- Music only: 70-100%
- Music under voiceover: 20-30%

### Step 6: Add Logo

1. Tap **Stickers** > **Add sticker**
2. Select your logo file from photos
3. Position in lower-left corner (safe zone)
4. Resize (drag corners)
5. Adjust opacity if needed (tap logo > Opacity)
6. Extend duration: Drag logo on timeline to cover entire video

**Logo specs:**
- PNG with transparent background
- Position: Lower-left corner
- Size: 10-15% of frame width
- Opacity: 50-70% for watermark, 100% for intro/outro

### Step 7: Add Captions (Auto)

1. Tap **Text** > **Auto captions**
2. Select language
3. Wait for processing
4. Review and correct any errors
5. Style captions to match brand

**Why captions matter:** 85% of Instagram videos watched without sound.

### Step 8: Export

1. Tap **Export** (top right arrow)
2. Settings:
   - Resolution: **1080p** (sufficient for Instagram)
   - Frame rate: **30fps**
   - Format: Best quality
3. Wait for export
4. Save to camera roll

## Brand Templates

### Creating a Master Template

1. Create a project with:
   - Intro (2-3 sec with animated logo)
   - Placeholder clips
   - Lower third text style
   - Outro with CTA (3-5 sec)
   - Music track
2. Save project
3. To reuse: Duplicate project, swap footage

### Elements to Standardize

| Element | Specification |
|---------|---------------|
| Intro | 2-3 sec, logo animation |
| Logo watermark | Lower-left, 50-70% opacity |
| Text font | [Brand font], bold |
| Text color | [Brand primary color] |
| Lower third | [Brand color] bar, white text |
| Outro | Logo + CTA, 3-5 sec |
| Music style | Consistent genre/mood |

### Template Types to Create

1. **Transformation Reveal** - Before/after with transition
2. **Progress Update** - Time-lapse with text overlays
3. **Educational Carousel** - Text-heavy, slow pacing
4. **Team Spotlight** - Name card, interview style
5. **Quick Tip** - Fast cuts, bold text

## Advanced Techniques

### Speed Ramping

**For dramatic reveals:**
1. Split clip at key moment
2. Tap first section > Speed > Normal
3. Tap second section > Speed > 0.5x (slow motion)
4. Add transition between

### Keyframe Animation

**For moving text/logos:**
1. Add text/sticker
2. Position playhead at start
3. Tap element > Keyframe (diamond icon)
4. Move playhead, reposition element
5. CapCut animates between points

### Color Correction

**Quick fixes:**
1. Tap clip > Adjust
2. Options:
   - Brightness: Fix dark footage
   - Contrast: Add punch
   - Saturation: Boost colors
   - Temperature: Fix color casts

**For consistency across clips:**
- Adjust first clip
- Tap **Copy** (adjustment settings)
- Select other clips > **Paste**

### Background Removal

**For layered effects:**
1. Tap clip > **Remove BG**
2. Wait for processing
3. Add background layer beneath

## Safe Zones (Critical!)

Instagram overlays UI elements on videos. Avoid placing important elements:

```
┌─────────────────────────────┐
│ ▓▓▓ AVOID TOP 250px ▓▓▓▓▓▓ │ ← Account name, follow button
│                             │
│     SAFE AREA FOR          │
│     IMPORTANT CONTENT      │
│                             │
│                        [X] │ ← Like/comment/share buttons
│ ▓▓▓ AVOID BOTTOM 400px ▓▓▓ │ ← Caption, audio info
└─────────────────────────────┘
```

**Logo placement:** Lower-LEFT corner (right side has buttons)

## Workflow for Managers

### Basic Edit (Manager Does)

1. Import footage
2. Trim to best clips (under 60 seconds total)
3. Arrange in logical order
4. Add basic text (project name, date)
5. Export draft
6. Upload to "FOR REVIEW" folder

### Final Polish (Finalizer Does)

1. Download from shared folder
2. Add intro/outro from template
3. Add logo watermark
4. Add licensed music
5. Add/style captions
6. Color correct for consistency
7. Export final
8. Move to "APPROVED" folder

## Common Issues & Fixes

| Problem | Solution |
|---------|----------|
| Text too small | Minimum 50pt for readability |
| Music too loud | Lower to 20-30% under voice |
| Video looks dark | Adjust > Brightness +10-20 |
| Shaky footage | Effects > Video > Stabilize |
| Wrong aspect ratio | Ratio tool > 9:16 for Reels |
| Logo disappears | Extend logo duration on timeline |
| Export fails | Free up phone storage |
