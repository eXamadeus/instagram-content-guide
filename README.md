## Quick Start (First Week)

### Day 1-2: Setup
1. Configure iPhone settings per [Equipment Guide](./guide/01-equipment-setup.md)
2. Download CapCut (free) on all manager phones
3. Set up shared Dropbox folder

### Day 3-4: First Content
1. Review [Filming Techniques](./guide/02-filming-techniques.md)
2. Capture footage using shot list
3. Create first edit using brand template

### Day 5: Launch
1. Get client permission ([Templates](./guide/10-templates.md))
2. Post first Reel
3. Review [Instagram Strategy](./guide/05-instagram-strategy.md) for optimization
